# ALU Design using Verilog

This project implements an Arithmetic Logic Unit (ALU) using Verilog HDL.

## Operations Supported
- Addition
- Subtraction
- AND
- OR
- XOR
- Shift operations

## Tools Used
- Verilog HDL
- ModelSim
- GTKWave

## Project Files
- alu.v : RTL implementation of ALU
- alu_tb.sv : Testbench for ALU verification
- sim.log : Simulation output
- waveform.vcd : Simulation waveform
